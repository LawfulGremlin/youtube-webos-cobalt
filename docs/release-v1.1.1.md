# v1.1.1

## Highlights

- Reduced the release package from about 46 MB to about 19 MB by removing
  Cobalt's runtime debug information during packaging.
- Added Spanish, French, Portuguese, Dutch and Polish translations to the
  YouTube Extras settings and SponsorBlock status labels.
- Included the visual-debug helper for diagnostics without enabling an overlay
  in normal release builds.

## Installation

Install `youtube.leanback.v4_1.1.1_arm.ipk` directly or use
`youtube.leanback.v4.manifest.json` with the webOS Homebrew Channel.
